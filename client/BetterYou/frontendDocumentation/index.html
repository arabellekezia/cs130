Parameters
$0 (Object)
Name Description
$0.style any
$0.iconName any
$0.startTime any
$0.activity any
$0.duration any
$0.caloriesBurned any

iconName (string) name of the MaterialCommunityIcons icon that will be fed into the Icon component
